## Learn the language of shipping

Vocab Vault is a mobile-first flashcard app designed to help aspiring developers master the vocabulary they need to build with AI coding tools. Whether you're using ChatGPT, Cursor, Lovable, or Bolt, understanding the right terms accelerates your journey from idea to deployed app.

**Perfect for:**

- Beginners using ChatGPT who want to learn more
- People starting with Lovable, Cursor, Bolt, or Replit
- No-coders transitioning to AI-assisted coding
- Anyone who wants to understand what developers are talking about

## Why vocabulary matters

When you're building with AI, knowing the right vocabulary is the difference between:
- "Make it look better" vs "Add hover states with 0.3s transitions"
- "Save my stuff" vs "Implement local storage persistence"
- "Put it online" vs "Deploy to production with CI/CD"

The more precise your language, the better your AI coding partner can help you build.

## Vocabulary categories

Vocab Vault organizes coding vocabulary into 24 focused categories:

| Category | Coverage |
|----------|----------|
| **Foundation** | Core concepts like MVP, Frontend, Backend, Deploy |
| **APIs** | Endpoints, authentication, API keys, webhooks |
| **Code & Files** | Repositories, branches, commits, packages |
| **Development** | IDEs, Terminal, debugging, localhost |
| **Git** | Version control fundamentals |
| **Cloud** | Infrastructure, hosting, serverless, Firestore |
| **UI/UX** | Design principles, components, wireframes |
| **CSS** | Styling fundamentals, Flexbox, Grid, animations |
| **AI / Prompts** | LLMs, tokens, prompt engineering, RAG |
| **No-Code** | Automation, workflows, integrations |
| **Money / Business** | SaaS, MRR, conversion, churn |
| **Tools** | Platform-specific vocabulary (Lovable, Cursor, Firebase) |
| **Security** | Authentication, encryption, vulnerabilities |
| **Testing** | QA, unit tests, integration tests |
| **Mobile** | App development, responsive design |
| **Data** | Databases, queries, schemas |
| **SEO** | Search visibility, meta tags, sitemaps |
| **Analytics** | Metrics, funnels, user tracking |
| **Architecture** | System design, patterns, scalability |
| **Hosting** | Deployment, domains, DNS |
| **AI Tools** | Specific AI coding assistants |
| **Package** | Dependencies, build tools, npm |
| **Shortcuts** | Keyboard shortcuts and CLI commands |
| **Debugging** | Error handling, stack traces, logging |

<Note>
  Many terms include ASCII visual examples to help you understand layout concepts, CSS properties, and system architecture patterns.
</Note>

## Built with modern tech

Vocab Vault is built with:

- **React** + **TypeScript** for type-safe component development
- **Vite** for lightning-fast dev server and builds
- **Tailwind CSS** for utility-first styling
- **Framer Motion** for smooth animations
- **shadcn/ui** for beautiful, accessible UI components
- **Capacitor** for native mobile support on Android
